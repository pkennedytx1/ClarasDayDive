Circular icon-only button for nav toggles, search submit, social links, and close actions.

```jsx
<IconButton label="Search" variant="solid"><SearchIcon/></IconButton>
<IconButton label="Open menu" variant="outline" size="lg"><MenuIcon/></IconButton>
```

Variants: `solid` (rose), `ink`, `outline`, `ghost`. Sizes `sm | md | lg` (36/44/52px — never below the 44px touch target for primary mobile actions). `label` is required for accessibility.
