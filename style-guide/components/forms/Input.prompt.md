Text field on paper-white with a teal focus ring; pill or rounded shape.

```jsx
<Input label="Email" type="email" placeholder="you@email.com" />
<Input pill placeholder="Ask Clara what to drink…" iconLeft={<SparkleIcon/>} />
<Input label="Name" error="Required" />
```

Use `pill` for hero/AI-search inputs, default rounded for forms. `label`, `hint`, `error` (turns the border rose), and `iconLeft` are supported.
